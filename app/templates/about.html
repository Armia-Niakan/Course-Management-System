About this project

This a Course Management System developed as a project for Guilan university advanced programming class.

User System

  • Three-tier role system (Student/Teacher/Admin)
  • Secure authentication with password hashing
  • Email-based password recovery
  • Environment variables for sensitive data protection

Scheduling

  • Interactive weekly calendar view
  • Automatic schedule conflict detection
  • Course capacity management

Administration

  • Comprehensive admin dashboard
  • System-wide user management
  • Detailed activity logs

Security Features

  • Environment variables for sensitive data
  • .env.example provided for easy setup
  • Werkzeug password hashing and session security

Technical Details

Technologies Used

Backend

Python Flask framework

json data storage

SMTP email integration

Frontend

HTML5

w3.css framework

Font Awesome icons

Requirements

Python 3.7+ runtime environment

All dependencies in requirements.txt

SMTP email service configuration

Proper .env file setup

Security Note: Always protect your .env file and change default admin credentials immediately
